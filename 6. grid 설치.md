## 2. LINUX.X64_193000_grid_home.zip 압축 해제
```bash
[oracle@psk1 ~]$ echo $GRID_HOME
/u01/app/grid/19c
[oracle@psk1 ~]$ mkdir -p /home/oracle/app/grid/19c
[oracle@psk1 ~]$ unzip LINUX.X64_193000_grid_home.zip -d /home/oracle/app/grid/19c
```
