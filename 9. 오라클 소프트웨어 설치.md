## 1. MobaXterm 환경 설정
<img width="806" height="540" alt="1" src="https://github.com/user-attachments/assets/50d7d57b-d1a7-4e45-bb0e-224bf1c35978" />

- 파일 압축 해제
```bash
[oracle@psk1 ~]$ cd $ORACLE_HOME
[oracle@psk1 19c]$ unzip /home/oracle/media/LINUX.X64_193000_db_home.zip -d $ORACLE_HOME
```

- runinstaller 실행
```bash
[oracle@psk1 19c]$ cd $ORACLE_HOME
[oracle@psk1 19c]$ ./runInstaller
```
